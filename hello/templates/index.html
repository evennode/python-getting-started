Guide for deploying Python apps to EvenNode

How to deploy your app

  • Clone this git repository
    git clone https://github.com/evennode/python-getting-started.git
  • Make the necessary changes
  • To deploy add EvenNode's remote git repo – see git deployment
    Or see FTP deployment
  • Deploy your code git push evennode master
  • Restart app if needed
